Instructions for Customizing Your Website Template

Welcome! This guide will help you customize different elements of your website template.

1. Styling and Appearance

Main Stylesheet

The main styling file is located at assets/style.css. Open this file to change the layout, typography, colors, and other styling options.

Example of modifying the body background color in style.css:

body {
    background-color: #f0f0f0;
}

Loader Styling

To modify the loader appearance, edit the style.css file in assets/. This file controls the animations and styles of the loading screen.

Example of changing loader background color in page-loader.css:

.loader {
    background-color: #ffffff;
}

2. JavaScript Files

Main JavaScript File

Your main JavaScript functions are stored in assets/script.js. Open this file to add new functions, modify existing functionality, or integrate plugins.

To add a new function in script.js, you can do something like this:

function newFeature() {
    console.log("New feature added!");
}

Bootstrap JavaScript

The Bootstrap JavaScript file is also located in assets/. This file provides interactive features like modals and dropdowns. Ensure that the jQuery library is loaded if you are using older Bootstrap versions.

3. Adding Content

To update text, images, and other content, open index.html and locate the specific sections you want to edit.

For example, to change a heading:

<h1>Welcome to My Website</h1>
